# IR 모듈 핀 맵핑 테이블

Arduino UNO R4 Minima (Renesas RA4M1) 기준. 이더넷 쉴드 점유 핀(D4, D10~D13)을 제외한
가용 핀을 디지털 → 아날로그 순서로 IR 모듈 1~15번에 1:1 배정한다.

## 이더넷 쉴드 점유 핀 (사용 불가)

| 핀 | 기능 |
|----|------|
| D4 | SD Card CS |
| D10 | SPI SS |
| D11 | SPI MOSI |
| D12 | SPI MISO |
| D13 | SPI SCK |

## IR 모듈 핀 맵핑 (1~15)

| Module ID | 핀 | 핀 번호 | 종류 | 비고 |
|-----------|-----|---------|------|------|
| 1 | D0 | 0 | 디지털 | Serial RX — 사용 시 시리얼 디버깅 불가 |
| 2 | D1 | 1 | 디지털 | Serial TX — 사용 시 시리얼 디버깅 불가 |
| 3 | D2 | 2 | 디지털 | |
| 4 | D3 | 3 | 디지털 | |
| 5 | D5 | 5 | 디지털 | |
| 6 | D6 | 6 | 디지털 | |
| 7 | D7 | 7 | 디지털 | |
| 8 | D8 | 8 | 디지털 | |
| 9 | D9 | 9 | 디지털 | |
| 10 | A0 | 14 | 아날로그 | |
| 11 | A1 | 15 | 아날로그 | |

| 15 | A5 | 19 | 아날로그 | I2C SCL과 공유 |

## 요약

- **점유(사용 불가):** D4, D10, D11, D12, D13 — 5핀
- **가용 디지털:** D0, D1, D2, D3, D5~D9 — 9핀 (Module 1~9)
- **가용 아날로그:** A0~A5 — 6핀 (Module 10~15)
- **총 가용:** 15핀 = IR 모듈 15개 1:1 배정

## 주의사항

- **D0/D1 (Module 1, 2):** Serial과 공유. 사용 시 시리얼 디버깅 불가 — 디버깅 시 점퍼 분리.
- **A4/A5 (Module 14, 15):** I2C(SDA/SCL)와 공유. I2C 주변장치 추가 시 Module 14·15 재배치 필요.
- 본 테이블은 `include/gpio_map.h`의 `IR_PINS[]` 배열과 일치한다.
