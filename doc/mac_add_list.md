# 아두이노별 MAC Address

`DE:AD:BE:EF` 4바이트를 고정 베이스로 사용하고, 마지막 2바이트를 `00:01`~`00:64`로 순차 증가시킨 100개 주소.

> 첫 옥텟 `DE`(=`11011110`)는 bit1=1(로컬 관리), bit0=0(유니캐스트)이라 실제 제조사 OUI와 충돌하지 않는다. 순차 부여라 같은 네트워크 내 중복 위험도 없음.
